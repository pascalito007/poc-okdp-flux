# OKDP — Open Kubernetes Data Platform

## Component Inventory

OKDP provides a modular, open-source data platform for Kubernetes. Each component can be installed independently or as part of a complete stack.

## Getting Started

1. **Explore components**: Browse the tables above to understand what OKDP offers
2. **Install infrastructure**: Follow the [okdp-prerequisites guide](https://github.com/OKDP/okdp-handbook/tree/main/modules/infrastructure/okdp-prerequisites/INSTALL.md) or install modules individually
3. **Install data platform**: Pick the modules you need and follow their install guides
4. **Try the examples**: Deploy [okdp-examples](https://github.com/OKDP/okdp-examples) for hands-on workflows
