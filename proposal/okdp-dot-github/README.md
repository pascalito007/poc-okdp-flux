# OKDP — Default Community Health Files

This repository contains the default community health files for the [OKDP](https://github.com/OKDP) organization.

GitHub automatically applies these files to all OKDP repositories that don't have their own version.

See [GitHub docs: Creating a default community health file](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).

## Files

| File                                | Purpose                                             |
| ----------------------------------- | --------------------------------------------------- |
| `CODE_OF_CONDUCT.md`                | Behavioral expectations (Contributor Covenant v2.1) |
| `CONTRIBUTING.md`                   | How to contribute to any OKDP repo                  |
| `SECURITY.md`                       | How to report security vulnerabilities              |
| `ISSUE_TEMPLATE/bug_report.md`      | Standardized bug report template                    |
| `ISSUE_TEMPLATE/feature_request.md` | Standardized feature request template               |
| `PULL_REQUEST_TEMPLATE.md`          | PR checklist template                               |

## Override

Any OKDP repo can override these defaults by creating its own version of the file.
For example, `okdp-handbook` has its own `PULL_REQUEST_TEMPLATE.md` with a module-specific checklist.
